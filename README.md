# local-messenger
мессенджер для тех, кто устал от блокировок. в поле "Ник" вписываете свое прозвище, в поле "IP" вписываете только ip вашего друга, приятеля и тд. если вы сами хотите создать чат, то нажимайте на "Запустить сервер" (и в приложении высветится ваш ip и ваш ник), а чужой собеседник должен будет вставить ваш ip. только тогда вы сможете общаться вместе
